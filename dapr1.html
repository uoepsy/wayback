Data Analysis for Psychology in R 1

Course description

Data Analysis for Psychology in R 1 (DAPR1) is a 1st year course in data analysis for psychological research, using the R software. This course will teach you how to collect, describe, plot, and analyse data, in order to extract meaningful insights.
