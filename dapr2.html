Data Analysis for Psychology in R 2

Course description

Data Analysis for Psychology in R 2 (DAPR2) is a 2nd year course in data analysis for psychological research, using the R software. This course will introduce you to statistical modelling beyond what is typically taught in an introductory statistics course, empowering you with tools to analyse richer data and answer a broader set of research questions of interest to you.
